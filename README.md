# Training Planner

## Автор
Шульга Илья

## Описание
Приложение для планирования и отслеживания тренировок с фильтрацией, сохранением данных в JSON и контролем версий через Git.

## Возможности
- Добавление тренировок (дата, тип, длительность).
- Фильтрация по типу и дате.
- Сохранение и загрузка данных из JSON.
- Проверка корректности ввода.

## Примеры использования
1. Запустите приложение: `python "Training Planner.py"`
2. Введите данные и нажмите «Добавить тренировку».
3. Используйте фильтры для поиска нужных записей.
4. Данные сохраняются автоматически.

## Технологии
- Python
- Tkinter (или другая GUI-библиотека)
- JSON
- Git
